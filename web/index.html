Weather_station_API v1.0

This api works by sending GET request to two routes, /api/today and /api/historic. A JSON file is returned with an array of objects named "data". Ex:
{ "data": [ { "temp": 30, "heat_index": 31.049081, "humidity": 50, "dew_point": 18.44087, "incidency_sun": 25, "precipitation": 5, "req_date": "2021-02-22", "req_time": "00:07:27" }, { "temp": 30, "heat_index": 31.049081, "humidity": 50, "dew_point": 18.44087, "incidency_sun": 25, "precipitation": 5, "req_date": "2021-02-22", "req_time": "00:07:29" } ] }

Route /api/today retrieves today's wheater data and /api/historic retrieves historic data between two times, both should be send as http options like this: /api/historic?date1=2021-02-01&date2=2021-03-20
